    Render(RenderArgs),
    AfterRender(AfterRenderArgs),
    Update(UpdateArgs),
    Idle(IdleArgs),

Adds render and update events to input events

Variants

Render(RenderArgs)

Render graphics.

AfterRender(AfterRenderArgs)

After rendering and swapping buffers.

Update(UpdateArgs)

Update the state of the application.

Idle(IdleArgs)

Do background tasks that can be done incrementally.

Input(I)

Input event.
