pub struct Rect {
    pub x: Range,
    pub y: Range,
}

Defines a Rectangle's bounds across the x and y axes.

This is a conrod-specific Rectangle in that it's designed to help with layout.

Fields

The start and end positions of the Rectangle on the x axis.
